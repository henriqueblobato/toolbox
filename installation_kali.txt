1.
Trocar repositório em:
    nano /etc/apt/sources.list

2.
Substituir por:
    deb http://http.kali.org/kali kali-rolling main contrib non-free
    # For source package access, uncomment the following line
    # deb-src http://http.kali.org/kali kali-rolling main contrib non-free

3.  
Checar endereço correto em:
    https://docs.kali.org/general-use/kali-linux-sources-list-repositories

4.    
Atualiar repositório e upgrade
    sudo apt-get update && sudo apt-get upgrade && sudo apt-get dist-upgrade

5.  
Na máquina virtual:
    Dispositivos > Inserir imagem de CD Adicionais

6.    
Arrastar VBoxLinuxAdditions.run para desktop

7.
Checar sua versão de kali com:
    uname -a

8.
Procurar e baixar o header respectivo em
    apt-get search headers | grep kali
    apt-get install linux-headers-<versao>-<arquitetura do processador>

9.
Atualiar repositório e upgrade
    sudo apt-get update && sudo apt-get upgrade && sudo apt-get dist-upgrade
    sudo reboot
